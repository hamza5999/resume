# Resume

This repository contains the source code and compiled PDF for my resume, built with LaTeX.

## Files

- `resume.tex` – Main LaTeX source file
- `resume.pdf` – Compiled resume
- `.gitignore` – Git ignore rules for LaTeX temporary files

## Build

Compile the resume using:

```bash
latexmk -pdf resume.tex
```

or

```bash
pdflatex resume.tex
```

## Notes

This repository is maintained for version control and personal use.
